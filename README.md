# Miko Manga Reader

Miko is a high-performance, Material 3 Expressive manga reader built with Kotlin and Jetpack Compose.

## Features
- **Expressive Design**: Large corner radii and dynamic color support.
- **Efficient Loading**: Memory-efficient image streaming using Coil.
- **Native Performance**: Built with native Kotlin for the best battery life.

## How to Build & Run
Since this project is designed for "no-IDE" development:

1. **Connect your physical Android phone** via USB.
2. **Enable USB Debugging** in Developer Options.
3. Open a terminal in the `miko` directory.
4. Run:
   ```bash
   ./gradlew installDebug
   ```

The app will be compiled and installed directly onto your phone.
